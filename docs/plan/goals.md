# ゴール

このファイルは、Sound Effect Generator で目指す到達点を共有するための場所です。

## プロジェクトの目的

[Stable Audio 3 Medium](https://huggingface.co/stabilityai/stable-audio-3-medium) を使い、テキストプロンプトから効果音を **ローカルで** 生成するデスクトップアプリを作る。

- クラウドに頼らず、自分の GPU で完結する
- 思いついた効果音をすぐ試し、気に入ったものを WAV で書き出せる
- 環境構築の手間を最小化し、プロジェクトフォルダ内で完結させる（システムを汚さない）

## 完成形のイメージ

- プロンプトと少数のパラメータ（長さ・ステップ数・CFG・シード）を設定して「生成キューに追加」
- 生成タスクはキューで順次処理され、結果はカードとして一覧に積まれていく
- 各カードで再生・WAV ダウンロード・条件の再利用ができる
- 生成結果はアプリを閉じても残る（永続化）
- ダブルクリック（`start.bat`）で起動でき、特別な知識がなくても使える

## 重視する価値

1. **ローカル完結・再現性** — オフライン動作、内蔵 Python、固定バージョンの依存
2. **手軽さ** — 起動・生成・保存が迷わずできる UI
3. **見やすさ** — ダークテーマで統一された、シンプルで詰め込みすぎない画面
4. **データを失わない** — 生成物とその条件を確実に保存する

## スコープ外（現時点）

- 音声編集（インペイント／継続生成）などの高度機能
- アプリのインストーラ配布（現状はリポジトリ + 内蔵ランタイムで起動）
